# What is this
Kaggleのマウスにおける社会的行動認識コンペティション用のリポジトリです。  
コンペティションは https://www.kaggle.com/c/MABe-mouse-behavior-detection にあります。

# How to join
1. Kaggleのユーザー登録を行いましょう。
2. 上記URLにある < Join Competition > をクリックしましょう。そうすれば、コンペティションに必要なデータのダウンロードが行えます。

# How to download
https://www.kaggle.com/competitions/MABe-mouse-behavior-detection/data の下のほうにある < Download All > をクリックすると 2GB のZipファイルのダウンロードが始まります。

# What you'll need
このコンペティションではPythonのコードの提出が必須となっていますので、Pythonの環境が必要です。  
KaggleのPython環境は3.11のようですが、作者はAnacondaの仮想環境でPython 3.12の環境を構築した後、KaggleのNotebookにコピー&ペーストで提出するようにしています。
